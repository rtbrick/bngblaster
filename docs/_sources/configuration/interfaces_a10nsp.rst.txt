    { "interfaces": { "a10nsp": [] } }

+-----------------------------------+----------------------------------------------------------------------+
| Attribute                         | Description                                                          |
+===================================+======================================================================+
| **interface**                     | | Parent interface/link name (e.g. eth0, ...).                       |
+-----------------------------------+----------------------------------------------------------------------+
| **qinq**                          | | Set outer VLAN ethertype to QinQ (0x88a8).                         |
|                                   | | Default: false                                                     |
+-----------------------------------+----------------------------------------------------------------------+
| **mac**                           | | Overwrite the A10NSP MAC address.                                  |
|                                   | | Default: `parent interface/link MAC address`                       |
+-----------------------------------+----------------------------------------------------------------------+
